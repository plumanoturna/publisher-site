Privacy Policy

Publisher — Multi-Channel Content Distribution Platform

Effective date: March 1, 2026

1. About Publisher Platform

This Privacy Policy applies to the Publisher app, a multi-channel content distribution platform. Publisher enables content creators to publish videos, shorts, images, and audio across multiple social media platforms from a single dashboard. This policy explains how Publisher handles user data.

2. Data Collection

The Publisher app does not collect, store, or share personal data from end users. Publisher is a content distribution tool that operates on behalf of the content creator who uses Publisher to manage their channels. Publisher does not track visitors, collect analytics, or maintain user profiles.

3. Authentication and Tokens

The Publisher app uses OAuth2 to authenticate with third-party platforms (YouTube, TikTok, Instagram, Pinterest). When a content creator connects their account to Publisher, the following applies:

  • Publisher stores OAuth2 access tokens and refresh tokens locally on the content creator's machine.
  • Publisher never transmits these tokens to any external server.
  • Publisher uses these tokens solely for the purpose of publishing content on behalf of the authenticated user.
  • Publisher tokens can be revoked at any time by the content creator through the platform's account settings or by disconnecting from Publisher.

4. Third-Party Services

Publisher distributes content through the following third-party platform APIs:

  • YouTube — Publisher uses the YouTube Data API v3 for video uploads and scheduling.
  • TikTok — Publisher uses the TikTok Content Posting API for video publishing.
  • Instagram — Publisher uses the Instagram API for Reels and image publishing.
  • Pinterest — Publisher uses the Pinterest API v5 for pin creation.
  • Spotify — Publisher generates RSS feeds for podcast distribution on Spotify.

Each platform has its own privacy policy. Publisher encourages users to review the privacy policies of these third-party services. Publisher is not responsible for the data practices of these platforms.

5. Content Data

Publisher processes content files (videos, images, audio, metadata) provided by the content creator for the sole purpose of publishing to the connected platforms. Publisher does not retain copies of published content beyond the local file system. Publisher does not modify content for purposes other than platform-specific format requirements.

6. Cookies and Tracking

The Publisher app does not use cookies, web beacons, or any tracking technologies. The Publisher website (this page) is a static site hosted on GitHub Pages and does not set any cookies or collect visitor data.

7. Data Security

Publisher takes data security seriously. All OAuth2 tokens stored by Publisher are kept on the local file system with standard file permissions. Publisher communicates with third-party APIs exclusively over HTTPS. Publisher does not operate any backend servers that process or store user data.

8. Children's Privacy

Publisher is a content distribution tool intended for use by content creators. Publisher does not knowingly collect data from children under the age of 13. Publisher is not directed at children.

9. Changes to This Policy

Publisher may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated effective date. Continued use of Publisher after changes constitutes acceptance of the updated policy.
